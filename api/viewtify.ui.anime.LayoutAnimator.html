LayoutAnimator

  • javafx.beans.value.ChangeListenerNumber
  • javafx.collections.ListChangeListenerNode

Animates an object when its position is changed. For instance, when additional items are added to a Region, and the layout has changed, then the layout animator makes the transition by sliding each item into its final place.
